TALOS HVAC LOGIN APPLICATION
=============================

A Java Swing application featuring a professional login interface and demo request form
for Talos HVAC hiring software.

FEATURES:
---------
✓ HVAC-themed picturesque design matching the Talos website
✓ Company logo and branding prominently displayed
✓ Login form with username/email and password fields
✓ "Forgot Password" and "Forgot Username" functionality
✓ Support contact information
✓ "Get Demo" button that opens a separate demo request form
✓ Demo form with validation for all required fields:
  - Full Name
  - Company Name
  - Email (with format validation)
  - Phone Number
  - Employee Count dropdown
✓ Form validation with error messaging
✓ Success confirmation with professional styling

COMPILATION:
------------
javac *.java

RUNNING:
--------
java TalosLoginApp

DESIGN ELEMENTS:
----------------
- Primary color: #1a5a3a (Talos green)
- Gradient backgrounds with subtle grid patterns
- Professional typography and spacing
- Responsive hover effects
- Modern rounded buttons and form elements
- Consistent with Talos website branding

REQUIREMENTS:
-------------
- Java 8 or higher
- Java Swing (included in standard JDK)

The application demonstrates professional UI design principles while maintaining
the HVAC industry focus and Talos brand identity.
